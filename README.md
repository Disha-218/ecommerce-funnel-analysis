E-Commerce Funnel & Conversion Analysis

Overview
Analyzed 10,000 user sessions from an e-commerce store using Python and SQL to identify where customers drop off in the purchasing process.

Key Insights
Overall Conversion Rate: 18.36% (1,836 purchases out of 10,000 sessions).
Main Bottleneck: 46% of users drop off between adding products to their cart and going to checkout.
Device Friction: Mobile users experience higher checkout drop-off compared to desktop users.

Recommendations
1. Simplify mobile checkout experience and add one-click payment options (Apple/Google Pay).
2. Set up automated abandoned cart emails for users who leave during checkout.

How to Run
1. Install Python packages: pip install pandas numpy plotly
2. Run script: python run_analysis.py
